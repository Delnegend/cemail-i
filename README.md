# Cemail-i

## Usage
```bash
$ cemail-i -h
Send an email from the CLI

Usage: cemail-i --sender <SENDER> --receiver <RECEIVER> --content <CONTENT> --password <PASSWORD>

Options:
  -s, --sender <SENDER>      Name <email@address>
  -r, --receiver <RECEIVER>  Name <email@address>
  -c, --content <CONTENT>    [subject] Content
  -p, --password <PASSWORD>  Google app password (https://myaccount.google.com/apppasswords, you need 2FA enabled)
  -h, --help                 Print help
```
